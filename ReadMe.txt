How to Use
============================================

1. Run the file dou.exe and wait for the update (internet connection must be available).
2. Turn off all you headphones.
3. Turn on the remote control WITHOUT the USB cable.
4. Go to update mode (Option > Configuration > Update > Confirm).
5. Connect the single USB cable between your PC and the remote control (don`t use the recharge cable with 3 mini USB).
6. Follow the further prompts of the program.
